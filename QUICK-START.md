# 🚀 LearnBite - Quick Start Guide

## ⚡ Start Using the App (1 Minute)

### Step 1: Open the App
```
http://localhost:5173/
```

### Step 2: Sign Up
- Click "Get Started"
- Enter name, email, password
- Click "Create Account"

### Step 3: Start Learning!
- Browse topics
- Select a lesson
- Complete and track progress
